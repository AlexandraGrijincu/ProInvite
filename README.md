# ProInvite – Platformă pentru generarea automată a invitațiilor personalizate
### Grijincu Alexandra-Ștefania

## Descriere
ProInvite este o aplicație web care permite utilizatorilor autentificați să creeze și să personalizeze invitații digitale pentru diverse tipuri de evenimente – precum nunți, botezuri sau aniversări. Utilizatorii pot completa un formular cu detaliile evenimentului (data, ora, locația, descrierea etc.), iar sistemul va genera automat o invitație pe baza unui template JSON importat de pe un repository GitHub extern.

## Functionalitati Principale
* **Asistent AI pentru Conținut:** Formularul include o integrare cu un **API de Inteligență Artificială**. Introduci câteva cuvinte cheie despre eveniment, iar sistemul generează automat o descriere creativă și personalizată.
* **Sistem de Template-uri Flexibil:**
    * **Modele Interne:** Selecție de template-uri predefinite disponibile direct în aplicație.
    * **Import Extern:** Posibilitatea de a importa template-uri noi prin utilizarea link-urilor directe către imagini de pe web.
* **Autentificare & Inregistrare** Pentru a putea genera invitatii.
* **Export PDF:** Generarea automată și descărcarea invitațiilor personalizate în format PDF.
